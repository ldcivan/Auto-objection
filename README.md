# Auto-objection
基于html和JavaScript的全自动異議あり！

### 使用方法
假设于服务器上之后，按一般网页访问即可

现在可自定义震动感知阈值、语音语种，也可以自定义拍桌后展示的图片、音频、台词

### Demo
在[Demo(GitHub，更稳定)](https://ldcivan.github.io/Auto-objection) 或者 [Demo(自有服务器，更新)](https://www.pro-ivan.com/game/objection/)
